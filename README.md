# FUDAMBIENT Landing Page

Landing page for FUDAMBIENT - Fundación para el Desarrollo Social, el Medio Ambiente, la Vivienda, la Recreación, el Deporte, el Comercio y el Empleo.

## Design Source

This landing page is built to match the Figma design exactly:
- **Figma File**: Page - FUDAMBIENT
- **Design URL**: https://www.figma.com/design/eKRpQN7Lyw82RLmOsTFSnM/Page---FUDAMBIENT?node-id=1-3

## Features

- ✅ Responsive header with scroll effect (90px default, 70px when scrolled)
- ✅ Hero section with animated title and CTA
- ✅ About section with styled frame
- ✅ Mission and Vision section with image and text
- ✅ Achievements section with statistics cards
- ✅ Specialties section with 8 specialty cards
- ✅ Projects slider with navigation
- ✅ Territory section with cards
- ✅ Footer with contact information
- ✅ Smooth scrolling navigation
- ✅ Interactive project slider with auto-play
